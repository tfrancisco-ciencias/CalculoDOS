Información general

Dinámica del curso

El curso está pensado para ser tomado a distancia. Por lo tanto será necesario tener acceso a un dispositivo para conectarse a internet para ver clases y poder subir archivos (principalmente pdf's).

Dos días a la semana se darán clases sincrónicas, dos días ayudantías (todo vía zoom, que serán grabadas) y un tercer día se publicarán ejercicios resueltos.

Para la evaluación se presentarán tres exámenes parciales y quizzes semanales. Se dejarán tareas las cuales no se califican pero sirven de práctica para los exámenes.

Plataformas

  • Google classroom

    • Se usará para organizar el material del curso: lista de ejercicios, videos, notas de clase, material de lectura. Además se usará esta plataforma para entregar sus evaluaciones (con hora y fecha límite).

  • Zoom

    • Se usará para sesiones sincrónicas. El link será publicado en Google classroom.

  • Página del curso.

    • Esta que estas leyendo. Su usará para publicar ejercicios resueltos.

  • Grupo de Telegram.

    • Se creara un grupo de Telegram para dudas. Detalles serán publicados en Google classroom.
